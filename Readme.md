# AI Data Quality Agent (dbt test generator)

Автоматизация создания тестов качества данных (Data Quality) с использованием LLM.

## Описание
Скрипт анализирует SQL-структуру таблиц и автоматически генерирует YAML-конфигурации для dbt. 
Он выходит за рамки простой проверки синтаксиса, предлагая логические тесты:
- Проверка на уникальность и отсутствие пустых значений (unique, not_null).
- Валидация диапазонов значений (positive_values).
- Проверка допустимых значений (accepted_values).

## Стек
- Python 3.11
- Ollama / Llama3
- dbt-style YAML schema

## Запуск
1. Запустите Ollama.
2. Поместите ваш SQL-файл в корневую папку.
3. Запустите скрипт: `python check_agent.py`.
4. Получите готовый `schema_tests.yml`.
